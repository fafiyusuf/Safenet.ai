HOW IT WORKS:
--------------
1. User visits /chat page
2. User types message to the counselor
3. Message sent to backend API (/api/chat)
4. Gemini AI (gemini-2.0-flash-exp) processes with therapeutic prompt
5. AI response displayed in chat interface
6. Conversation context maintained (last 10 messages)
7. User can continue chatting or use emergency buttons
